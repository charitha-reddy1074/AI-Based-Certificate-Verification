## How to Upload This Data in Admin Portal

### Steps:

1. **Navigate to Admin Dashboard**
   - Go to the application and login as admin

2. **Click "Bulk Upload (CSV)" Button**
   - Located in the Admin Dashboard header
   - Opens the bulk upload dialog

3. **Download Sample Template** (Optional)
   - Click "Download Sample CSV" to get the format
   - Or use the provided `sample_certificates.csv`

4. **Select CSV File**
   - Browse and select `sample_certificates.csv`
   - File validation checks:
     - ‚úÖ All required columns present
     - ‚úÖ Data format validated
     - ‚úÖ Duplicate roll numbers checked
     - ‚úÖ Required fields populated

5. **Review Upload Preview**
   - See all 20 records to be uploaded
   - Check for any errors/warnings
   - Confirm data is correct

6. **Click "Upload Certificates"**
   - System processes all 20 records
   - Each certificate gets:
     - Unique Certificate ID
     - QR Code
     - Blockchain Hash
     - Transaction Hash
     - Block Hash
     - Previous Hash

7. **View Success Results**
   - Successfully Uploaded: 20
   - Failed: 0
   - Completion time: 2-3 seconds

8. **Certificates Now Visible in Admin Portal**
   - All 20 certificates appear in dashboard
   - Searchable by roll number, name, branch
   - Filterable by batch year and branch
   - Sortable by CGPA, name, date issued
