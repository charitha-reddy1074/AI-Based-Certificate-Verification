# AI Based Decentralised Academic Credential Verification System

AI Based Decentralised Academic Credential Verification System is a decentralized certification verification platform built on blockchain technology.
It enables institutions to issue tamper-proof digital certificates.
Individuals can own and share their credentials securely.
Employers and verifiers can instantly validate certificates without intermediaries.
All records are immutable, transparent, and trustless.
AI Based Decentralised Academic Credential Verification System reduces fraud and manual verification costs.
The system supports academic, professional, and skill-based certifications.
Privacy is preserved through cryptographic proofs.
It brings trust and efficiency to credential verification.
